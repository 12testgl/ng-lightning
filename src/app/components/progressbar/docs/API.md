# <ngl-progress-bar>

| Property | Description | Type | Default |
| -------- | ----------- | ---- | ------- |
| `[value]` | the percentage value | number | |
| `[size]` | size of the progress bar | 'x-small' \| 'small' \| 'medium' \| 'large' | 'medium' |
| `[variant]` | variant appearance of the progress bar | 'circular' | 'medium' |
