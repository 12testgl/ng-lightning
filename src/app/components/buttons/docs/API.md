# `[nglButton]`

### Input

  * `nglButton?: 'success' | 'neutral' | 'brand' | 'destructive' | 'inverse'`: Buttons theme.

# `[nglButtonState]`

### Input

  * `nglButtonState: boolean`: Whether button is active or not.

### Output

  * `nglButtonStateChange: EventEmitter<boolean>`: the requested state change.
